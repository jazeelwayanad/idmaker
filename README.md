# ID Maker

Bulk ID Card Production System built with Next.js, Electron, TypeScript, and SQLite.

## Prerequisites

- Node.js 18+
- npm

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Package Desktop App

```bash
npm run dist
```
